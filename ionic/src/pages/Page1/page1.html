Page1 Welcome to Convertigo Ionic 2 template Menu based Layout

This starter project comes with simple menu based layout for apps that are going to primarily use a Menu. Take a look at the src/pages/ directory to add or change page, update any existing page or create new pages.

Convertigo Angular Framework (CAF)

You can also take a look at the src/app/app.component.ts and src/pages/Login/login.html to understand CAF.


You will discover routing table concept, automatic variable mapping and server sequence calls from HTML templates.
